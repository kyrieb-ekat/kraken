How to train a model

  1. 1 Upload a CSV

    Drop a Cantus Database export (.csv) into the upload zone on the left. The app will read each row's folio label, manuscript image URL, and transcription text, grouping them by page.

  2. 2 Fetch page images

    Click View Folios next to your dataset, then Download Images. The app will download each manuscript page from the URLs in the CSV. If a download fails (e.g. restricted access), use the Upload Image button that appears next to that folio, or upload a ZIP of images all at once.

    Tip: filenames in the ZIP should match the folio label (e.g. 001v.png).

  3. 3 Review & correct transcriptions Go to Review →

    Switch to the Review tab. Select your dataset and a folio, then click Segment Page. Kraken will detect individual lines of text on the page.

    Each detected line appears with a suggested transcription drawn from the CSV. Because one CSV row (chant) can span several physical lines, you may need to:

    • Edit a line's text directly in the text box.
    • Split a line if a new chant begins mid-line — click "Split" and then click the split point on the image.
    • Merge two lines if a single chant was split across them — select both and click "Merge Selected".
    • Delete a line if Kraken detected a ruling or decoration rather than text.
    • Add a line by dragging a box on the page image if Kraken missed one.

    When a line's transcription is correct, tick Confirmed. Use Confirm All once a whole folio looks good.

  4. 4 Compile & train Go to Training →

    Switch to the Training tab. First, click Compile GT to package all confirmed lines into a training archive. Then fill in a run name and epoch count, choose an optional base model to fine-tune from, and click Start Training.

    The live log panel streams Kraken's output in real time. You can stop a run at any time and resume later from a checkpoint.

  5. 5 Download your model Go to Models →

    Once training finishes, go to the Models tab. Your trained .mlmodel file will be listed there and available to download. You can also select it as a base model for future fine-tuning runs.

Keyboard Shortcuts

Lines

A Add a new line by drawing a polygon
S Split selected line (then click split point on image)
M Merge selected line (then click line to merge with)
Delete or Backspace Delete selected lines

Editing

Click Select/deselect line on image
Edit button Edit polygon shape (drag vertices, insert/delete points)
Ctrl/Cmd+Z Undo last action

Navigation

Esc Cancel current mode (add, split, merge, edit)
Scroll wheel Zoom in/out
Click+Drag Pan the image
